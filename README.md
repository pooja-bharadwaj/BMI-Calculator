# BMI Calculator

## Description
This is a simple Body Mass Index (BMI) Calculator web application.  
It calculates BMI using the user's height (in cm) and weight (in kg) and displays the BMI value along with the corresponding weight condition.

## Features
- Takes height and weight as user input
- Calculates BMI instantly
- Displays BMI value up to two decimal places
- Shows weight condition (Underweight, Normal, Overweight)
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript
